CryptojackerFinder.
Описание работы программы.

При нажатие кнопки "Старт" программа анализирует загрузку ЦП всеми процессами и ищет процессы которые потребляют больше CPU чем указано в поле "Порог использования CPU (%)".
Далее, найденные процессы добавляются в таблицу и анализируется. Если такие процессы не найдены, то анализ заверщается. При анализе процессов проверяются "Среднеквадратичное 
отклонение ЦП", "Потребление памяти", "Скорость передачи по сети", "Количество обращений к библиотекам из CryproApi в минуту". Конкретно проверяющиеся параметры сравниваются с 
заданными граница значений. Если все значения больше значения границ, то процесс считается криптоджекером и окращивается в зеленый. В остальных случаях процесс окращивается в 
красный цвет и криптоджекером не считается. Внизу указывается время анализа. При заверщение анализа процессы, которые выделены зеленым, считаются криптоджекерами.
